Etapa 1 - Análisis del Sistema - Diseño de interfaces

## 1. Nombre del Proyecto
Sistema de predicción longitudinal de problemas visuales

## 3. Problema que se quiere resolver
En la práctica clínica actual, el optómetra promedio debe atender diariamente una alta
afluencia de pacientes, esto se hace mediante métodos tradicionales. La dependencia a
procesos mecánicos genera un alto porcentaje de margen de error, comprometiendo la
precisión diagnostica y eficacia del personal. Para solucionar esto, en el mercado ya existen
varios sistemas para poder automatizar dichos procesos. Sin embargo, estas suelen
presentar barreras críticas, como lo son costos elevados, diseños exclusivos para
ecosistemas de marcas específicas, o tecnologías excesivamente complejas (Deep learning,
Machine Learning).

## 4. Usuarios del Sistema
- Optómetras
- Pacientes
- Administradores

## 5. Entrada, Proceso y Salida del Sistema

### Entrada
- Datos personales del paciente
- Fecha de la consulta
- Observaciones del Optómetra
- Historial optométrico previo
- Resultados de pruebas complementarios (tipografía corneal, fondo de ojo, etc.)
- Factores ambientales

### Proceso
- Organización estructurada del historial clínico.
- Comparación longitudinal de resultados entre consultas.
- Aplicación de algoritmos matemáticos básicos (regresión lineal, promedios móviles, análisis de tendencia).
- Cálculo de intervalos de confianza para estimaciones.
- Identificación de variaciones significativas en parámetros visuales.
- Generación automática de alertas ante posibles cambios patológicos.
- Visualización gráfica del progreso o deterioro de variables clínicas.

### Salida
- Historial clínico digital organizado.
- Gráficas evolutivas de parámetros visuales.
- Proyección matemática del posible avance de una condición ocular.
- Intervalos de confianza sobre los valores estimados.
- Alertas de riesgo basadas en cambios estadísticamente relevantes.
- Reportes imprimibles para el profesional o el paciente.

## 6. Alcance del Sistema

### Incluye:
- Registro y almacenamiento de historiales clínicos optométricos.
- Análisis longitudinal de datos clínicos.
- Generación automática de gráficas evolutivas.
- Proyección matemática básica de posibles progresiones.
- Cálculo de intervalos de confianza.
- Sistema de alertas por cambios significativos.
- Optimización del flujo de trabajo clínico.
- Exportación de reportes en formato digital.

### No incluye:
- Diagnóstico médico automático.
- Sustitución del criterio profesional del optómetra.
- Cirugías o tratamientos automatizados.
- Integración directa con equipos de marcas específicas (a menos que se desarrolle en futuras versiones).
- Uso de modelos avanzados de Deep Learning o Machine Learning complejos.
- Gestión hospitalaria completa (facturación, inventario, etc.), si no forma parte del enfoque principal.

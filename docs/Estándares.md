# UI KIT

Este documento define los elementos visuales del sistema.
---

## 1. TIPOGRAFÍA

- Fuente principal: `News Gothic BT`
- Fuente secundaria: `Roboto`

### Jerarquía tipográfica

- **Título 1 (H1):** `32px` – Bold  
- **Título 2 (H2):** `24px` – SemiBold  
- **Subtítulo:** `18px` – Medium  
- **Párrafo grande:** `16px` – Regular  
- **Párrafo pequeño:** `14px` – Regular  
- **Cita:** `14px` – Italic  
- **Botón:** `16px` – Medium  

---

## 2. ICONOS

**Estilo de iconos:** `Filled (tipo dashboard moderno / Material)`

- **Icono activo:** color `#9FEAE4`  
- **Icono inactivo:** color `#6B7280`  

**Ejemplos:**
- Calendario
- Usuario activo  
- Icono inactivo  

---

## 3. COLORES

### Paleta principal

- **Primario:** `#9FEAE4`  
- **Secundario:** `#1F2937` 
- **Acento:** `#00747C`

### Extras (importante por tu UI oscuro)

- **Background:** `#111827` 
- **Texto principal:** `#FFFFFF`
- **Texto secundario:** `#9CA3AF`

---

## 4. BARRAS (NAVEGACIÓN)

### Barra superior (Header)

**Elementos:**
- Logo + nombre: *OptoHelp*  
- Icono de usuario / cerrar sesión  
- Fondo oscuro (`#111827`)  

---

### Barra inferior (Bottom Navigation)

**Secciones:**
- Inicio  
- Pacientes  
- Citas  
- Reportes  

---

## 5. TARJETAS 


**Elementos:**
- Icono (calendario / usuario)  
- Título 
- Fondo oscuro (`#1F2937`)  
- Hover: leve cambio de color  

---

### Tarjeta pequeña (Estadísticas)

**Elementos:**
- Texto 
- Número grande  
- Color destacado: `#9FEAE4`  
- Fondo oscuro  

---

## 6. BOTONES Y CONTROLES

### Botones

- **Botón primario (activo)**  
  Fondo: `#9FEAE4`  
  Texto: negro  

- **Botón primario (inactivo)**  
  Fondo: gris (`#374151`)  
  Texto: gris claro  

- **Botón secundario (activo)**  
  Fondo: transparente  
  Borde: `#9FEAE4`  

- **Botón secundario (inactivo)**  
  Borde: gris  

- **Botón flotante (FAB)**  
  Fondo: `#00747C`  
  Icono: blanco  

---

### Controles

- Slider (para rangos de datos)  
- Filtros (por paciente, fecha o demás)  
- Switch (activar/desactivar seguimiento)  

---

## 7. FORMULARIOS

**Elementos:**
- Input de texto  
- Input de contraseña  
- Checkbox  
- Ingreso de datos

### Estilo

- Fondo: `#1F2937`  
- Texto: blanco  
- Borde: gris  

### Estados

- **Normal:** borde gris  
- **Activo:** borde `#9FEAE4`  
- **Error:** borde rojo (`#E53935`)
